## How to Use

```bash
git clone https://github.com/sensq/bastion-guacamole
cd bastion-guacamole
./init.sh
docker-compose up -d
./start_browser.sh
```

Access to `http://<hostip or hostname>:8080/guacamole`

login `guacadmin/guacadmin`

Connect!
